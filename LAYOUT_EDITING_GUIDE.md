# 레이아웃 편집 가이드

## 방법 1: 브라우저 개발자 도구 사용 (가장 빠름)

1. **개발자 도구 열기**
   - `F12` 키 누르기
   - 또는 우클릭 → "검사" / "Inspect"

2. **요소 선택**
   - 개발자 도구 왼쪽 상단의 요소 선택 아이콘 클릭 (또는 `Ctrl+Shift+C`)
   - 화면에서 편집하고 싶은 요소 클릭

3. **스타일 수정**
   - 오른쪽 패널의 "Styles" 탭에서 CSS 속성 수정
   - 예: `margin`, `padding`, `gap`, `top`, `left` 등

4. **수정된 값 확인**
   - 개발자 도구에서 수정한 CSS 값을 복사
   - 해당 파일의 해당 부분을 찾아서 수정

## 방법 2: 코드에서 직접 수정

관리자 페이지의 주요 레이아웃 요소들:

### 필터 탭 영역
- 파일: `app/(main)/admin/page.tsx`
- 위치: 약 1155번째 줄 근처
- 클래스: `flex items-center justify-between mb-4`

### 버튼 간격
- `gap-2` → `gap-4` (간격 늘리기)
- `px-4 py-2` → `px-6 py-3` (버튼 크기 조정)

### 테이블 간격
- `px-6 py-4` → `px-8 py-5` (셀 간격 조정)

## 방법 3: Tailwind CSS 유틸리티 클래스

- 간격: `gap-2`, `gap-4`, `gap-6`, `gap-8`
- 여백: `mb-4`, `mb-6`, `mb-8`, `mt-4`, `mt-6`
- 위치: `top-0`, `left-0`, `right-0`, `bottom-0`
- 크기: `w-full`, `w-1/2`, `max-w-4xl`, `max-w-6xl`
